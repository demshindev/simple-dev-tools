# SimpleDevTools

A collection of useful developer tools in one place. All tools work completely in the browser without any backend - no server required, all processing happens client-side.

## Features

- JSON Formatter & Validator
- XML Formatter
- Base64 Encoder/Decoder
- JWT Encoder/Decoder
- UUID/GUID Generator

- Markdown Preview
